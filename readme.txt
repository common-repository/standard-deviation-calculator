A WordPress plugin to calculate standard deviation.

== Description ==

[Standard deviation calculator](https://www.standarddeviationcalculator.io) is a free calculator plugin that finds the standard deviation of an entered set of data. Using this plugin, you can allow users to find the variance, Standard Deviation, Differences, Sum, and Square of Differences for free.

== Features ==

1. Through this plugin users on your website can find following values of given set of data 

    *Sample & population (default "sample")
    *Sample Variance
    *Sample Standard Deviation
    *Population Variance
    *Population Standard Deviation
    *Differences
    *Square of Differences

2. You can also change the calculator background from admin panel.

3. It is very easy to use, You just have to use shortcode [standard-deviation] to show the calculator on webpage.

== Installation and Use ==

This section describes how to install and use this plugin on your dashboard.

* Upload the plugin files to your directory, or install the plugin through the Wordpress plugin directory and it is totally free.
* Activate the plugin through 'Plugins' screen in the WordPress dashboard.
* Now you can see the standard deviation calculator menu on wordpress admin dashboard.
* To add it on front end you just have to add short code there [standard-deviation].

== It's not working ==

- Step 1. Check if your Plugin is activated.
- Step 2. Deactivate all plugins and reactivate Standard deviation calculator.
